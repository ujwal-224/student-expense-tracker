## Tech Stack
HTML5- Structure of Application
CSS3- Styling and Layout
JavaScript (Vanilla)- Application Logic and Interactivity

## Storage of Data:
Browser localStorage: Long-term expense data storage without a backend


## Analytics and Perspectives:
JavaScript-based reasoning for summaries based on categories
AI prompt design-inspired rule-based insight generation

## Use of AI:
Quick engineering for insight logic and expense classification
The `prompts/` directory contains the documentation for prompts.
To ensure simplicity and reproducibility, no external AI APIs are utilized.

## Implementation:
It operates locally in any contemporary web browser.
GitHub Pages can be used for deployment (optional).

## Development Methodology:
Frontend-focused design
Dependency-free and lightweight
Completely repeatable throughout the system
