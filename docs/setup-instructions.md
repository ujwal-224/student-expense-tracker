## Setup Instructions
The Student Expense Tracker is a lightweight web application built using HTML, CSS and Javascript. It does not need any backend server or framework.

## Steps to run the project
1. CLone the GitHub repository
   git clone <repository-url>
2. Navigate to the project directory
   cd student-expense-tracker
3. Open the application
   open src/index.html in any web browser such as chrome, edge or firefox.

The application will run locally in the browser and all expense data will be stored using the browsers local storage.
